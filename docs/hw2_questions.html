Replication of Angrist & Lavy (1999)

Using Maimonides’ Rule to Estimate the Effect of Class Size on Scholastic Achievement

Author

Fancheng Xia

Published

April 22, 2026

Introduction

How much does class size actually affect student learning? This seemingly simple question has proven surprisingly hard to answer. The core problem is endogeneity: schools often assign struggling students to smaller classes, and wealthier districts tend to have both larger schools and better outcomes. A naive comparison of large vs. small classes conflates the effect of class size with these confounding factors.

Angrist and Lavy (1999) solve this identification problem using an elegant natural experiment rooted in a 12th-century rabbinical rule. Maimonides’ Rule, still enforced in Israeli public schools today, caps class size at 40 students. When a grade’s enrollment hits 41, the cohort must be split into two classes; at 81, into three; and so on. This creates sharp, predictable discontinuities in class size as a function of enrollment — discontinuities that have nothing to do with student ability or family background.

The key insight is that a school with 40 students in a grade will have one class of 40, while a school with 41 students will have two classes of about 20. These two schools are likely very similar in every other way, yet experience a dramatic difference in class size driven purely by an administrative rule. Angrist and Lavy use the predicted class size from Maimonides’ Rule as an instrumental variable, allowing them to isolate the causal effect of class size on test scores.

Formally, the predicted class size (instrument) is:

\[f_{sc} = \frac{e_s}{\lfloor (e_s - 1)/40 \rfloor + 1}\]

where \(e_s\) is the beginning-of-year grade enrollment in school \(s\). This paper is a landmark application of the regression-discontinuity (RDD) design in economics.

Main Analysis

Data

The data cover all Jewish public elementary schools in Israel tested in 1991. The unit of observation is the class. Outcomes are average reading (verbal) and math composite scores (scaled 1–100). The key control variable is tipuach, the percent of students from disadvantaged backgrounds (the PD index).

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.ticker as ticker
import statsmodels.formula.api as smf
from IPython.display import display

# Load 4th grade data
df = pd.read_stata("files/final4.dta")

# Fix scores > 100 (coding artifact in original data)
df["avgverb"] = np.where(df["avgverb"] > 100, df["avgverb"] - 100, df["avgverb"])
df["avgmath"] = np.where(df["avgmath"] > 100, df["avgmath"] - 100, df["avgmath"])

# Maimonides' Rule predicted class size
df["func1"] = df["c_size"] / (np.floor((df["c_size"] - 1) / 40) + 1)

# Set scores to missing if no test takers
df.loc[df["verbsize"] == 0, "avgverb"] = np.nan
df.loc[df["mathsize"] == 0, "avgmath"] = np.nan

# Apply sample restrictions (identical to original Stata do-file)
df = df[(df["classize"] > 1) & (df["classize"] < 45) & (df["c_size"] > 5)]
df = df[(df["c_leom"] == 1) & (df["c_pik"] < 3)]

print(f"Sample size: {len(df):,} classes")
print(f"Mean verbal score: {df['avgverb'].mean():.1f}")
print(f"Mean math score:   {df['avgmath'].mean():.1f}")
print(f"Mean class size:   {df['classize'].mean():.1f}")

Table II (Columns 7 & 10) — Naive OLS

Before introducing the instrument, we estimate a simple OLS of test scores on actual class size with no controls. Standard errors are clustered by school (schlcode), following Moulton (1986).

def ols_cluster(outcome, regressors, data, cluster):
    cols = [outcome] + regressors + [cluster]
    d = data[cols].dropna()
    formula = outcome + " ~ " + " + ".join(regressors)
    m = smf.ols(formula, data=d).fit(
        cov_type="cluster", cov_kwds={"groups": d[cluster]}
    )
    return m

m_t2c7  = ols_cluster("avgverb", ["classize"], df, "schlcode")
m_t2c10 = ols_cluster("avgmath", ["classize"], df, "schlcode")

rows = []
for label, m, var in [
    ("Col 7: Verbal ~ Class Size", m_t2c7, "classize"),
    ("Col 10: Math ~ Class Size",  m_t2c10, "classize"),
]:
    rows.append({
        "Specification": label,
        "Class Size Coef": f"{m.params[var]:.3f}",
        "Std. Error":      f"({m.bse[var]:.3f})",
        "N":               int(m.nobs),
        "R²":              f"{m.rsquared:.3f}",
    })

t2 = pd.DataFrame(rows).set_index("Specification")
display(t2)

Table III (Panel A, Columns 7, 9, 11) — Reduced-Form RDD

Instead of regressing scores on actual class size, we now regress scores on the Maimonides’ Rule instrument (func1), controlling for PD (tipuach) and optionally enrollment (c_size). This is the reduced-form of the RDD strategy.

m_t3c7  = ols_cluster("avgverb", ["func1", "tipuach"],           df, "schlcode")
m_t3c9  = ols_cluster("avgverb", ["func1", "tipuach", "c_size"], df, "schlcode")
m_t3c11 = ols_cluster("avgmath", ["func1", "tipuach"],           df, "schlcode")

rows = []
for label, m, var in [

    ("Col 11: Math   ~ func1 + PD",                m_t3c11, "func1"),
]:
    rows.append({
        "Specification": label,
        "func1 Coef": f"{m.params[var]:.3f}",
        "Std. Error": f"({m.bse[var]:.3f})",
        "N":          int(m.nobs),
        "R²":         f"{m.rsquared:.3f}",
    })

t3 = pd.DataFrame(rows).set_index("Specification")
display(t3)

Something Additional — Why Does the Sign Flip? Visualizing Bias vs. Causation

The naive OLS and the reduced-form RDD tell completely opposite stories. This section makes that contrast concrete and explains why the RDD estimate is the credible one.

Why the RDD Estimate Is Credible

The sign reversal — from positive (OLS) to negative (RDD) — is not a mystery. It reflects two distinct sources of bias in the naive OLS:

1. Between-school selection bias. Larger schools in Israel are disproportionately located in prosperous urban areas. These schools also tend to have larger class sizes and higher-scoring students. This creates a spurious positive correlation between class size and test scores across schools.

2. Within-school selection bias. Even within the same school, principals may assign weaker students to smaller, remedial classes. This further inflates the OLS coefficient.

The Maimonides’ Rule instrument sidesteps both problems. The rule assigns predicted class size based purely on enrollment thresholds — whether a cohort falls just above or just below a multiple of 40. This threshold is effectively random with respect to student ability or family wealth. A school with 41 students is nearly identical to a school with 39 students in every way except that the former must split into two classes of ~20 while the latter has one class of 39.

By controlling for smooth functions of enrollment (the c_size covariate) and the PD index, and using only the discontinuous variation in func1 as the source of identification, the RDD isolates the causal channel: class size → test scores, with all confounders held fixed.

The result: a ten-student reduction in predicted class size is associated with roughly a 0.85–0.89 point increase in 4th grade reading scores — a credible, causally-identified estimate that the naive OLS would have hidden entirely.
